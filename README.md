## vue-init

> A boilerplate project for vue. 一个vue样板项目。

### 特性

* 支持vue全家桶
* 代码检测
* es6转es5
* 代码打包压缩
* 静态资源打包压缩
* webpack页面配置剥离，仅需专注于开发
* 配置了丰富的webpack插件，便于开发
* 支持scss/sass使用
* 多页面打包

### 如何运行

* 安装依赖包

  ```bash
    $ npm install
  ```

* 开启开发环境服务

  ```bash
    $ npm run start
  ```

### 如何构建项目

* 构建开发环境代码

  ```bash
    $ npm run dev
  ```

* 构建生产环境代码

  ```shell
    $ npm run prod
  ```
