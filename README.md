### 如何运行

* 安装依赖包

  ```shell
    $ npm install
  ```

* 开启开发环境服务

  ```shell
    $ npm run start
  ```

### 如何构建项目

* 构建开发环境代码

  ```shell
    $ npm run dev
  ```

* 构建生产环境代码

  ```shell
    $ npm run prod
  ```
